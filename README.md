###  How to Contribute

üõ†Ô∏è Let's make this contest journey as smooth as possible for you:üõ†Ô∏è

1. **Fork this Repository** üç¥

   Click the "Fork" button at the top right corner of this repository to create your personal fork.

2. **Clone Your Fork** üì•

   ```bash
   git clone https://github.com/codemacrocosm/Web-a-Thon-20.git
   ```

3. **Move Inside the Repository**: Open your terminal and navigate to the repository folder üìÇ

4. **Create a New Branch**: It's time to make your mark. Create a new branch üå±

5. **Bring Your Vision to Life**: Explore the `CONTRIBUTE.md` file to discover various ways you can contribute to this web development extravaganza üöÄüí°

6. **Commit Your Masterpiece**: Once you've poured your creativity into your work, commit your changes üé®

7. **Create Separate Files for Resources and Suggestions**: To share your resources and suggestions, create separate markdown files:
   
    - For resources, in the file named `resources.md` and add your recommended resources there üìö

    - For suggestions, in the file named `suggestions.md` and add your suggestions there üí¨


8. **Create a Pull Request**: Visit your GitHub repository and click the "Compare & Pull Request" button. Submit your pull request, and feel free to add a personal message üöÄüí¨
